# KAOS Roadmap

> Ver el estado detallado y marcado en [`../ROADMAP.md`](../ROADMAP.md).

## Alpha (completada)
- Foundation repository
- Bootstrap
- CLI
- Runtime (EventBus, Storage)
- SDK
- Discord Connector (gateway + backfill)
- Resume Agent
- LLM Provider (OpenAI-compatible / GitHub Models / Anthropic)
- Persistencia PostgreSQL + cache de resúmenes
- Suscripciones persistidas (`kaos subscribe` / `kaos run`)

## Beta (en curso)
- Configuración de providers (`kaos providers`)
- Scheduler (`kaos schedule`)
- Knowledge Graph
- Multi-workspace
- Odoo & GitHub connectors
- Dashboard

## 1.0
- Stable SDK
- Distributed runtime
- Plugin marketplace
- Production documentation
